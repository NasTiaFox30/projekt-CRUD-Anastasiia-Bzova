# Database Scripts
Scripts for manage DataBase:

```bash
# Interaktywne czyszczenie bazy danych
npm run cleanup

# Czyszczenie lokalnej bazy danych
npm run cleanup:local

# Czyszczenie zdalnej bazy danych
npm run cleanup:remote

# Tryb interaktywny
npm run migrate

# Migracje dla lokalnej bazy danych
npm run migrate:local

# Migracje dla zdalnej bazy danych
npm run migrate:remote

# Pełny reset lokalnej bazy danych
npm run reset

# Pełny reset zdalnej bazy danych
npm run reset:remote
```
